# 每日签到提醒系统

## 功能介绍

一个基于Python开发的每日签到提醒桌面软件，具有以下功能：

1. **用户注册**：支持输入用户名和邮箱进行注册
2. **今日签到**：点击绿色签到按钮记录签到时间
3. **签到历史查询**：查看最近30天的签到记录
4. **自动提醒**：连续2天未签到时，通过邮件发送提醒
5. **数据持久化**：使用SQLite数据库存储用户信息和签到记录
6. **跨平台支持**：适配Windows和macOS系统

## 技术栈

- **开发语言**：Python 3.7+
- **GUI框架**：Tkinter
- **数据库**：SQLite
- **邮件服务**：smtplib/email
- **定时任务**：schedule

## 安装说明

### 1. 安装Python环境

确保您的计算机已安装Python 3.7或更高版本。您可以从[Python官网](https://www.python.org/)下载并安装。

### 2. 安装依赖包

打开命令行终端，进入项目目录，执行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

### 3. 配置邮件信息

1. 复制或重命名`config.ini.example`为`config.ini`（如果不存在）
2. 编辑`config.ini`文件，填写您的发件人邮箱和授权码：

```ini
[Email]
sender_email = 您的邮箱@qq.com  # 或其他邮箱
sender_password = 您的邮箱授权码
```

**获取授权码教程**：

- **QQ邮箱**：
  1. 登录QQ邮箱网页版
  2. 点击右上角"设置" → "账户"
  3. 下滑找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
  4. 开启"SMTP服务"
  5. 点击"生成授权码"，按照提示操作获取16位授权码

- **163邮箱**：
  1. 登录163邮箱网页版
  2. 点击右上角"设置" → "POP3/SMTP/IMAP"
  3. 开启"SMTP服务"
  4. 设置客户端授权密码，保存后记住该密码

## 运行软件

### 方法1：直接运行主程序

在项目目录下执行以下命令：

```bash
python main.py
```

### 方法2：双击运行（Windows）

直接双击`main.py`文件即可运行（前提是已关联Python程序）。

## 使用说明

### 1. 用户注册

1. 在软件顶部的用户信息区域，输入您的用户名和邮箱
2. 点击"保存用户信息"按钮完成注册

### 2. 今日签到

1. 确保已完成用户注册
2. 点击中间的绿色圆形"今日签到"按钮
3. 签到成功后，状态会显示为"今日已签"

### 3. 查看签到状态

- **今日签到状态**：显示"今日已签"或"今日未签"
- **连续签到天数**：显示当前连续签到的天数
- **本月签到进度**：显示本月已签到的天数进度条

### 4. 查看签到历史

在软件底部的签到历史区域，可以查看最近30天的签到记录，包括签到日期和状态。

配置文件`config.ini`包含以下配置项：

| 配置项 | 说明 | 默认值 |
|-------|------|--------|
| sender_email | 发件人邮箱地址 | 空 |
| sender_password | 发件人邮箱授权码 | 空 |

## 日志文件

软件运行过程中产生的日志会保存在`sign_in_reminder.log`文件中，包括：

- 定时任务执行情况
- 邮件发送记录
- 错误信息和异常

## 常见问题

### 1. 邮件发送失败怎么办？

- 检查`config.ini`中的邮箱配置是否正确
- 确保已开启SMTP服务并获取了正确的授权码
- 检查网络连接是否正常
- 查看日志文件获取详细错误信息

### 2. 软件启动失败怎么办？

- 确保已安装所有依赖包
- 检查Python版本是否符合要求（3.7+）
- 查看命令行输出的错误信息

### 3. 如何重置数据？

直接删除项目目录下的`sign_in.db`文件，重启软件即可重新创建数据库。
